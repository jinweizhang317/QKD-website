# QKD FieldLab Launch Event Website

This repository contains the source code for the **QKD FieldLab Launch Event webpage**.

The website is a simple static site built with HTML and CSS and hosted using **GitHub Pages**.

## ✏️ How to Edit the Website

Anyone with access can update the site.

### Option 1 — Edit directly on GitHub (easiest)

1. Open the file (e.g. `index.html`)
2. Click the ✏️ **Edit** button
3. Make your changes
4. Click **Commit changes**

👉 Changes will go live automatically after ~1 minute.

---

### Option 2 — Edit locally (recommended for bigger changes)

1. Clone the repository:

```
git clone https://github.com/jinweizhang317/QKD-website.git
```

2. Open the folder on your computer
3. Edit files using VS Code or another editor
4. Push changes:

```
git add .
git commit -m "Update website"
git push
```

---

## 🔄 Collaboration Guidelines

* Always pull the latest version before making changes:

```
git pull
```

* Keep commit messages clear:

  * "Update agenda"
  * "Add partner logos"
  * "Fix layout issue"

* Avoid editing the same section at the same time to prevent conflicts

## 🚀 Deployment

This site is deployed using **GitHub Pages**.

* Source: `main` branch
* Folder: `/ (root)`

Any change pushed to `main` will automatically update the live website.
